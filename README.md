# Skenion CI

Reusable GitHub Actions workflows and composite actions for Skenion repositories.

This repository intentionally replaces an org-level .github automation repository for Skenion-specific CI rules.

## Status

Bootstrap repository for the Skenion project. Implementation follows the public architecture and release rules defined in [EchoVisionLab/skenion](https://github.com/echovisionlab/skenion).
